Detailed Description

Partition base class.

When instanced with a GDP, the Partition will initialize to a range of filtered offsets, suitable for interating over primitives or points, for example by passing a reference to GA_Iterator.

You need to derive new partitions from this class and pass the appropriate partition::Filter to the constructor.

See Also
partition::Filter, GA_Range, GA_RangeTypeInterface, GA_Iterator.
