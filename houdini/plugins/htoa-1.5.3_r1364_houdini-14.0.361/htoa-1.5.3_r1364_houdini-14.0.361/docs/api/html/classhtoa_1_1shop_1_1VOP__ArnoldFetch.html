std::vector< VOP_ArnoldFetch::OutputData > htoa::shop::VOP_ArnoldFetch::userDataOutputs ( ) const
private

Get outputs from userdata.

The node's user data contains a string that is a serialized version of the node's outputs. This method unserializes them and returns a vector of OutputData. This methods needs to be const to be used in the const methods outputLabel() and getNumVisibleOutputs().

We need to stash the outputs configuration in the node because the number of outputs depends on the target node pointed by the fetch node. When a scene containing a fetch node is loaded, it is possible that it is instanced before the target node, and yet we need to be able to run methods like getOutputTypeSubclass() before that.

The outputs configuration is cached in the _outputs data member and when it is the _is_outputs_initialized data member is set to true.
