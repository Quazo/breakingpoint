Return a defaulted parameter metadata string.

Note
As the order metadata string can be very long (~730 characters for the standard shader) and because of limitations of the metadata parser as of Arnold 4.0.13.0, we can split string metadata information in a series of string metadata named "foo", foo2", "foo3", etc. that will get concatenated.
