Detailed Description

This class handles exporting all or part of the Maya scene to Arnold, for rendering, exporting as an ass file or any other use.

It has methods to export individual objects or whole scenes, and tracks the list of translators created for each Maya node export.

Once CArnoldSession::Export() is called, the DAG hierarchy is traversed and CDagTranslators are found and exported for all relevant Maya nodes. Those translators in turn call and CArnoldSession::ExportShader() as they require, which triggers dependency graph evaluation and the generation of CNodeTranslators.

In IPR mode, the racked translators allows the scene to be quickly and incrementally retranslated as changes occur to previously translated Maya objects.

void CArnoldSession::SetExportCamera ( MDagPath  camera)

Set the camera to export.

If called prior to export, only the specified camera will be exported. If not set, all cameras will be exported, but some translators may not be able to fully export without an export camera specified. To address this potential issue, this method should be called after a multi-cam export, as it will cause all translators for which CNodeTranslator::DependsOnExportCamera() returns true to be updated.
